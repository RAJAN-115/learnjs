Maharashtra Ladli Bahen Yojana e-KYC Guide

Yahan par step-by-step dekho ki kaise tum apna e-KYC complete kar sakte ho bina kisi tension ke!

1. Portal Access Karo

  1. Browser mein jao aur visit karo:
    ladakibahin.maharashtra.gov.in
  2. Homepage pe dikhe “e-KYC” banner pe click karo.

2. Aadhaar & Captcha Enter Karo

  1. Apna 12-digit Aadhaar number daalo.
  2. Screen pe jo captcha code dikh raha hai, woh type karo.

3. OTP Verify Karo

  1. Send OTP button pe click karo.
  2. Aadhaar-linked mobile pe aaya hua 6-digit OTP daalo.
  3. Submit pe click karke verification complete karo.

4. Details Confirm Karo

  1. Screen pe tumhara Name, Address, aur Samagra ID aayege.
  2. Agar sab sahi hai toh Continue ya Proceed pe click karo.

5. Documents Upload Karo

Agar portal maange toh clear scans/photos upload karo:

  • Recent passport-size photograph
  • Domicile Certificate / Ration Card / Voter ID
  • Income Certificate (agar applicable ho)
  • Bank Account Details (IFSC code ke sath)

6. Declaration Complete Karo

Required checkboxes tick karo aur self-declaration accept karo.

7. Final Submission

  1. Sare entries ek baar dhyan se dekh lo.
  2. Submit pe click karo.
  3. Jo confirmation message aaye, uski screenshot le lo ya URN note kar lo.
Required Documents Checklist:
  • Aadhaar Card
  • Samagra (Family) ID
  • Aadhaar-linked Mobile Number
  • Recent Passport-Size Photograph
  • Domicile / Ration Card / Voter ID
  • Income Certificate (agar applicable ho)
  • Bank Account Details (with IFSC)

Important Notes

  • Process bilkul free hai—kisi ko payment mat karo.
  • Sirf official portal use karo.
  • Deadline se pehle e-KYC complete karo taki benefits uninterrupted mile.
